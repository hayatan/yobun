# タスク完了時チェックリスト

## コード変更後
1. **ドキュメント更新**
   - 関連するREADME.mdを更新
   - AGENTS.mdとの整合性を確認
   - .cursor/rules/yobun.mdcとの整合性を確認

2. **スキーマ変更時**
   - マイグレーションファイル作成（`sql/*/migrations/`）
   - SQLiteとBigQuery両方に対応

3. **店舗追加時**
   - `src/config/slorepo-config.js`を編集

4. **SQL変更時**
   - `sql/AGENTS.md`を参照

## Gitコミット
- コミットメッセージは**日本語**
- プレフィックス使用: `[追加]`, `[修正]`, `[変更]`, `[削除]`, `[リファクタ]`, `[ドキュメント]`

例:
```
[追加] スロレポのスクレイピング機能を実装
[修正] データ取得時のエラーハンドリングを改善
```

## テスト・動作確認
- 変更後は`make build`でイメージ再ビルド
- `make run-docker`でサーバー起動して動作確認
- エラーがないか確認

## 注意
- 本プロジェクトには自動テスト/リンター/フォーマッターは設定されていない
- 手動での動作確認が必要
