# 狙い台一覧 使い方ガイド

## 📌 このシートについて

このシートは、過去のデータを分析して「高設定になりやすい台」を推薦するためのリストです。
299種類の戦略を組み合わせて、各台番を総合的に評価し、優先度ランクを付けています。

---

## 🎯 優先度ランクの見方

| ランク | 意味 | 期待値 | 推奨度 |
|--------|------|--------|--------|
| **5** | 最優先 | 勝率63%、機械割113% | ⭐⭐⭐⭐⭐ 最優先で狙う |
| **4** | 高優先 | 勝率60%、機械割110% | ⭐⭐⭐⭐ 優先的に狙う |
| **3** | 優先 | 勝率56%、機械割108% | ⭐⭐⭐ 狙う候補 |
| **2** | やや優先 | 勝率50%、機械割106% | ⭐⭐ 参考程度 |
| **1** | 参考 | - | ⭐ 参考程度 |
| **0** | 対象外 | - | - 対象外 |

**推奨**: 優先度ランク **3以上**（`priority_rank >= 3`）の台を狙うことをおすすめします。

---

## 📊 各カラムの意味

### 基本情報

- **target_date**: 推奨日付（この日に狙う台）
- **machine_number**: 台番号
- **priority_rank**: 優先度ランク（5=最高、0=対象外）
- **rank**: その日のスコア順位（1位が最高）

### スコア関連

- **total_score**: 総合スコア（高いほど良い）
- **top1_ratio**: TOP1スコアとの比率（0.95以上が目安）
- **top1_score**: その日のTOP1のスコア（参考）

### 評価指標

- **match_count**: 該当戦略数（複数の戦略で推奨されているほど信頼性が高い）
- **weighted_payout_rate**: 重み付け機械割（例: 1.085 = 108.5%）
- **weighted_win_rate**: 重み付け勝率（例: 0.583 = 58.3%）
- **rms**: RMSスコア（機械割と勝率のバランス）
- **frequency_bonus**: 出現頻度ボーナス（複数戦略で推奨されている場合に高くなる）
- **dual_high_bonus**: 複合ボーナス（機械割と勝率の両方が高い場合に1.1倍）

## 💡 実際の使い方

### ステップ1: 今日の日付を確認する

1. シートの **target_date** 列を見て、今日の日付の行を探します
2. 日付で並び替える場合は、**target_date** 列を選択して「データ」→「範囲を並び替え」で日付順に並び替えます

### ステップ2: 優先度ランクで絞り込む

1. **priority_rank** 列を見て、**3以上**の行を探します
2. 並び替える場合は、**priority_rank** 列を選択して「データ」→「範囲を並び替え」で降順（大きい順）に並び替えます
3. 条件付き書式設定を使えば、優先度ランク3以上を色分けして見やすくできます

### ステップ3: 台を選ぶ

1. **優先度ランク5の台**があれば、それを最優先で狙う
2. **優先度ランク4の台**があれば、それを優先的に狙う
3. **優先度ランク3の台**があれば、それを狙う候補にする

### ステップ4: 複数台ある場合の選び方

同じ優先度ランクの台が複数ある場合は、以下の順で判断してください：

1. **total_score** が高い台を優先（既に高い順に並んでいます）
2. **match_count** が多い台を優先（複数の戦略で推奨されているので信頼性が高い）
3. **top1_ratio** が0.95以上の台を優先（より信頼性が高い）
4. **weighted_payout_rate** と **weighted_win_rate** の両方が高い台を優先

### ステップ5: 見やすくするコツ

- **条件付き書式設定**を使って、優先度ランク3以上を色分けすると見やすくなります
  - 優先度ランク5: 赤色
  - 優先度ランク4: オレンジ色
  - 優先度ランク3: 黄色
- **フィルタビュー**を作成すれば、優先度ランク3以上だけを表示できます

---

## ⚠️ 注意事項

- **このシートは過去データに基づく予測です**。100%確実ではありません。
- **優先度ランク3以上でも、必ずしも高設定になるとは限りません**。
- **複数の台が推奨されている場合は、自分の判断も加えて選んでください**。
- **データは手動で更新しています**。最新のデータか確認してから台を選んでください。

## ❓ よくある質問

**Q: 優先度ランク5の台が複数ある場合は？**
A: `total_score` が高い順に並んでいるので、上位の台を優先してください。また、`match_count` が多い台も信頼性が高いです。同じ優先度ランクの場合は、`match_count` が多い台を選ぶことをおすすめします。

**Q: 優先度ランク0の台は狙わない方がいいですか？**
A: はい。優先度ランク0は「対象外」なので、狙わないことをおすすめします。優先度ランク3以上の台を狙ってください。

**Q: `top1_ratio` が0.95未満でも優先度ランク3以上なら狙えますか？**
A: はい、狙えます。ただし、`top1_ratio` が0.95以上の場合の方が、より信頼性が高いです。可能であれば、`top1_ratio` が0.95以上の台を優先してください。

**Q: 今日の日付のデータが見つからない場合は？**
A: データは手動で更新しています。最新のデータが反映されていない可能性があります。管理者に確認してください。

**Q: データを並び替える方法は？**
A: 並び替えたい列のヘッダーをクリックして、「データ」→「範囲を並び替え」を選択します。優先度ランクで並び替える場合は、**priority_rank** 列を選択して降順（大きい順）に並び替えてください。

**Q: 条件付き書式設定の使い方は？**
A: 優先度ランクの列を選択して、「表示形式」→「条件付き書式設定」を選択します。例えば、優先度ランク5のセルを赤色、優先度ランク4をオレンジ色、優先度ランク3を黄色に設定すると見やすくなります。
