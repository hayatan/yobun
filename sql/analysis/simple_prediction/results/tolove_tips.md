## 注意点

### 連敗台を狙うときの心構え
- 連敗台は「反発」を期待するが、**低設定の可能性もある**
- 特日（LINE告知、月末、8のつく日）との組み合わせなら強い
- 通常日の連敗台はあまり期待できない

### 避けるべき組み合わせ
- **0のつく日 × 連勝/連敗** → 全体的にマイナス
- **1のつく日** → 日付自体がマイナス効果
- **9のつく日 × 5連勝** → 91.74%で大幅マイナス（サンプル少だが危険）

## 実践的な使い方

### ステップ1: 日付を確認
1. LINE告知日？ → **最優先で打つ**
2. 月末？ → **狙い目**
3. 8のつく日？ → 連敗台があれば狙い目
4. 1のつく日？ → **避ける**
5. それ以外？ → 連勝台を探す

### ステップ2: 前日までの勝敗を確認
- **5連勝台** → どの日でも狙える（特に通常日でも有効）
- **5連敗台 + 特日** → 最強の組み合わせ
- **3連敗台 + 8のつく日** → 108.81%の最強パターン

### ステップ3: 優先順位
1. 8のつく日 + 3連敗/5連敗
2. LINE告知 + 5連敗
3. 月末 + 3連敗
4. 5連勝台（どの日でも）
5. LINE告知 + 3連勝
